Vectors

The foundation of data representation in machine learning

🔵 What is a Vector?
A vector is an ordered list of numbers representing a point or direction in space. In machine learning, each data sample is represented as a feature vector.
v = [v₁, v₂, ..., vₙ]
For example, a house could be: v = [1200, 3, 2, 250000] (area, bedrooms, bathrooms, price)
📐 Vector Operations
Key operations you'll use constantly in ML:
Magnitude: ‖v‖ = √(v₁² + v₂²)

Dot Product: a·b = Σ aᵢbᵢ

🔢 What is a Matrix?
A matrix is a rectangular array of numbers arranged in rows and columns. In ML, matrices represent datasets, transformations, and weight layers in neural networks.
A = [[a₁₁, a₁₂], [a₂₁, a₂₂]]
⚙️ Why Matrices Matter in ML
Every neural network layer performs: output = W × input + b

Where W is a weight matrix. A dataset of 1000 samples with 10 features is a 1000×10 matrix. Matrix operations let us compute predictions for all samples simultaneously (vectorization).

⚖️ Law of Large Numbers
As n → ∞, the empirical frequency converges to the true probability. Try flipping 10 vs 10,000 times to see this in action. This is why ML models need large datasets!

🔮 Bayes' Theorem
Bayes' theorem lets us update our beliefs when new evidence arrives. It's fundamental to probabilistic ML models like Naive Bayes classifiers.
P(A|B) = P(B|A) × P(A) / P(B)

Normalization & Standardization

Scale your features before feeding data to ML models

📊 Enter Your Data
ORIGINAL MIN-MAX (0–1) Z-SCORE VISUALIZATION
📐 Min-Max Normalization
x' = (x - x_min) / (x_max - x_min)
Scales data to [0, 1]. Use when: features have known bounds, distribution doesn't need to be Gaussian, algorithms sensitive to magnitude (KNN, SVM, Neural Nets).
📊 Z-Score Standardization
x' = (x - μ) / σ
Centers data at 0 with std dev 1. Use when: data is approximately Gaussian, algorithms assume zero-centered data (logistic regression, PCA), outliers are present.
🤔 Which to Use?
Min-Max — images (pixel values 0–255), bounded features, when you know the min/max.

Z-Score — general purpose, when outliers matter, when you expect Gaussian data, for PCA.
