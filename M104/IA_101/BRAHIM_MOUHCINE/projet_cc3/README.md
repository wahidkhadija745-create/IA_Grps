MBI Chatbot — AI Data Chatbot (NYC Taxi Dataset)
Description

MBI Chatbot est une application web développée par Mouhcine & Brahim Intelligence (MBI). Elle permet d’interagir avec un dataset réel (NYC Taxi) via un chatbot intelligent capable de répondre en langage naturel.

Le projet combine interface web moderne et intelligence artificielle pour proposer une analyse conversationnelle des données.

Objectifs du projet
Concevoir un chatbot intelligent basé sur des données réelles
Permettre l’analyse de données via une interface conversationnelle
Développer une interface web moderne et interactive
Intégrer un modèle d’intelligence artificielle
Fonctionnalités
Chatbot interactif en temps réel
Réponses basées sur un dataset réel
Interface utilisateur moderne avec mode clair/sombre
Animations dynamiques en JavaScript
Architecture simple et modulaire
Saisie de clé API

Dataset

Le projet utilise un dataset de trajets de taxi (NYC Taxi) contenant :

durée des trajets
distances
zones géographiques
informations temporelles

Ce dataset permet de répondre à des questions analytiques sur les données.
